Personal Projects

  • minOS: kernel that can run on the i386 architecture. Features full interrupt handling facilities, device driver support, an extensible macro-based unit testing framework, and a full suite of memory management interfaces, including buddy-slab page frame allocators, dynamic kernel memory heaps, and kernel object cache allocators.
  • veriscan: FreeBSD daemon that uses a loadable kernel module to securely cache and periodically scan kernel memory in order to detect and then subsequently patch DKOM and system kernel memory hooks. Demonstrated to work against rootkits employing phantom system call tables and even kernel object hooks.
  • churchOS: HolyC-based operating system cloned off Terry A. Davis' TempleOS with the design goal of improved security and better kernel memory interfaces
